# ☕ Sales Trend and Time-Based Performance Analysis for Afficionado Coffee Roasters

## 📌 Project Overview
This project analyzes transaction-level coffee sales data from **Afficionado Coffee Roasters** to uncover temporal sales patterns and customer purchasing behavior. Using **Power BI**, the project transforms raw sales data into interactive dashboards that help identify peak sales hours, busiest days, revenue trends, and store-wise performance.

The insights generated from this analysis can help management optimize staff scheduling, improve operational efficiency, and make informed business decisions.

---

## 🎯 Problem Statement
Afficionado Coffee Roasters has access to transaction-level sales data but lacks a consolidated view of sales performance over time.

Without proper analytics, managers face challenges such as:
- Overstaffing during slow hours
- Understaffing during peak periods
- Inefficient operational planning
- Inconsistent customer experience

This project provides data-driven insights to support better operational and staffing decisions.

---

## 🎯 Objectives

### Primary Objectives
- Analyze overall sales trends throughout 2025
- Identify the busiest and slowest days of the week
- Determine peak transaction hours

### Secondary Objectives
- Compare sales performance across store locations
- Support staff scheduling decisions
- Improve operational planning using data-driven insights
